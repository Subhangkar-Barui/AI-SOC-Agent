# Frontend

React/Vite frontend for the SOC dashboard, log workflows, alerts, agent pairing, connected agents, network devices, live traffic, analytics, and reports.

## Run Locally

```powershell
npm install
Copy-Item .env.example .env
npm run dev
```

## Environment Variables

- `VITE_API_BASE_URL`
- `VITE_AGENT_DOWNLOAD_URL`
